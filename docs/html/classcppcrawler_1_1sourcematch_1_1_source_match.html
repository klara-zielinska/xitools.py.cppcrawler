def cppcrawler.sourcematch.SourceMatch.end (   self,
*  groups 
)

Returns the indices of the ends of the substrings matched by subgroups.

If there is a single argument, the result is a single number; if there are multiple arguments, the result is a tuple with one item per argument. No arguments is equivalent to passing 0. The 0 argument means the entire match. If a passed group exists, but did not contribute to the match, -1 is returned.
