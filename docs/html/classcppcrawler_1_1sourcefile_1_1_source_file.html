Detailed Description

Class for storing C++ source files.

Currently only Windows files are supported.

The line break sequences <new line> are removed from the code on loading, however their positions are stored.

The code positions taken and returned by methods are by default positions in the original code in public methods, and, respectively, positions in the internal code representation in internal methods (names starting with _).

Some public methods accept the int switch that makes them operate on internal positions when set True.

Finds the first occurence of a pattern in the search scopes.

Parameters
patRegular expression - can be from re or from its extension regex module - or a regular expression compiled with regex.compile. If skipBlocks=True, it has to be prepared with SourceFile.makeSkipBlocksPat or be equivalent.
skipBlocksIf True, excludes blocks (any { ... }) that start in the examined scopes from the search.
scopeTagIf True, a pair (SourceRegexMatch, tag) is returned, where tag is the tag of the scope where the occurrence was found.
excludeClipsIf True, clipped ranges are excluded from the search (see SourceFile.isClipped).
Returns
Depending on scopeTag either SourceRegexMatch, or (SourceRegexMatch, tag), or None.
Remarks
By exclusion we mean that a match cannot start in a given range, however it can start before and overlap it.

Tries to set the search scope to the body of a class, a struct or a collection of either.

If the method succeed, the tags of the new scopes are tuples (kind, name) or (kind, name, tag), where kind is either "class", "struct" or "unit", name is the name and tag is the tag of the scope that was searched.

Parameters
kindEither "class", "struct" or "union", or any combination of these separated with |, or "*" (any).
nameString or regular expression.
scopesIf present, the search is performed in the given <scope> collection, where <scope> is a tuple (begin, end) or (begin, end, tag), begin, end are positions in the code and tag is anything.
skipBlocksIf True, excludes blocks (any { ... }) that start in the examined scopes from the search.
scopeTagIf True, the new scopes contain the tag part.
Returns
True if the scopes were set, otherwise False.
