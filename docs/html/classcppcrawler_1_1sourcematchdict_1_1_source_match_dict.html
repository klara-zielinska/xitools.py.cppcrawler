Detailed Description

Dictionary class from SourceFile to SourceMatch list that stores matches for further processing.

It inherits from dict where keys are SourceFile and values are lists of either SourceMatch or pairs ( SourceMatch , tag) – tag may be anything. The lists are positionally ordered. The None key contains pairs (None, tag) or None values – depending on the tagging – that correspond to missed searches. This approach is chosen to allow unifor proccessing of all items.

Remarks
The iter operator returns an iterator that skips the None key. The iterators returned by items(), keys() and values() include all items.
