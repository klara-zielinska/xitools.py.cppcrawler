Finds one occurrence of each given pattern in sources.

Parameters
tpatsList of regex patterns or pairs (pattern, tag).
sourcesSources to be searched. A *<source>*, a collection of *<source>*, or a dict { <source> : <scope> collection }, where *<source>* is a path or a SourceFile, *<scope>* is a tuple (begin, end) or (begin, end, tag), begin, end are positions in the file and tag is anything. The dict can be a SourceScopeDict.
skipBlocksIf True, excludes blocks (any { ... }) that start in the examined scopes from the search.
perScopeIf True, finds an occurrence for each pattern in each given scope.
tagFilterSubstring of "ps" or None. "p" - include pattern tag, "s" - include scope tag, None - auto detect from tpats, sources.
Returns
SourceMatchDict of results. Missing patterns are stored in the SourceMatchDict.missing. If perScope=True, there is an entry for each missing pattern under the corresponding source key, otherwise the entry is placed under the key None.

Searches for method/function declarations.

No namespace support.

Parameters

sourcesSources to search. A glob file path (e.g., "*.h"), a SourceFile, or a list of either.
returnTypeType of the returned value:
  • ‘'s’- @ref SourceMatchDict with prototypes being tags, *'c'- dictionary{ <class name> : list[( <prototype>, (@ref SourceFile, @ref SourceMatch)|None )] }, *'l'- list[ ( <prototype>, ((SourceFile ,SourceMatch )|None ) ].

Collection of matches aligned with prototypes.
