Finds one occurrence of each given pattern in sources.

Parameters
tpatsList of regex patterns or pairs (pattern, tag).
sourcesSources to be searched. A <source>, a collection of <source>, or a dict { <source> : <scope> collection }, where <source> is a path or a SourceFile, <scope> is a tuple (begin, end) or (begin, end, tag), begin, end are positions in the file and tag is anything. The dict can be a SourceScopeDict.
skipBlocksIf True, excludes blocks (any { ... }) that start in the examined scopes from the search.
perScopeIf True, finds an occurence for each pattern in each given scope.
tagResultControls if the returned SourceMatchDict is tagged. Tags are evaluated by tagFunc.
tagFuncFunction that returns a tag for a result match. The first argument is a pattern that was matched or a pair (pattern, tag) - depending on the type of tpats. The second is a scope in which the match occurred - either (begin, end) or (begin, end, tag). If scopes were not given, the default (None, None) is passed.
Returns
A SourceMatchDict of results. Missing patterns are presented in 2 ways. If perScope=True, there is an entry for each missing pattern under the corresponding source key, otherwise the entry is placed under the key None. If tagResult=True, the entries are pairs (None, tag), otherwise they are None-s. This approach is used to allow a uniform iteration over all elements in the dict.

Searches for method/function declarations.

No namespace support.

Parameters
protsCollection of method/function prototypes in the normal form (cf. Syntax module).
sourcesSources to search. A glob file path (e.g., "*.h"), a SourceFile, or a list of either.
returnTypeType of the returned value:
  • 's' - SourceMatchDict with prototypes being tags,
  • 'c' - dictionary { 'class name' : list[( 'prototype', (SourceFile, SourceMatch)|None )] },
  • 'l' - list[ ( 'prototype', (SourceFile, SourceMatch)|None ) ].
Returns
Collection of matches. Matches align to found prototypes - e.g., fooBar( int i ).
