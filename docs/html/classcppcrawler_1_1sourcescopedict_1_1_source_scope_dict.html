Detailed Description

Dictionary class that stores source file scopes.

It inherits from dict where keys are SourceFile and values are lists of either pairs (start, end) or triples (start, end, tag) – start, end are original source file positions (see SourceFile) and tag may be anything. The lists are positionally ordered.

A user is obligated to maintain the order in value lists if he manipulates them directly. For safe insertion see SourceMatchDict.insert.
